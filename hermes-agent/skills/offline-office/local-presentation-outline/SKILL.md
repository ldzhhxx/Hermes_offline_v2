# 离线演示文稿大纲

## 适用场景
- 把报告、需求、项目资料转成 PPT 大纲。
- 生成每页标题、要点、讲稿和备注。

## 输出格式
每页使用以下结构：
- 页码 / 标题
- 核心信息
- 页面要点
- 建议图表或版式
- 讲稿备注

## 离线约束
- 不调用在线设计服务或素材库。
- 如果需要生成 pptx，优先使用仓库已有工具或已安装依赖；不要临时联网安装。
