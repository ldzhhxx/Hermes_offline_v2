# 离线报告生成

## 适用场景
- 汇总本地资料形成报告。
- 生成管理层摘要、项目复盘、风险评估、实施方案。

## 推荐结构
1. 摘要
2. 背景与目标
3. 已有材料依据
4. 关键发现
5. 分析与判断
6. 风险与限制
7. 建议与下一步

## 质量要求
- 明确区分“材料事实”和“推断建议”。
- 引用本地文件名或章节作为依据。
- 对缺失信息列出待补充清单。

## 离线约束
- 不进行公网搜索；报告依据只来自用户输入和本地文件。
